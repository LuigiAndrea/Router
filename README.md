Angular Route
===

#### Route Demos
Various demos about the Angular Route.

Some of the features used in the project:

- Guards
- Resolvers
- Lazy Loading
- Preloading modules
- Observables
- Promises
- CanActivate/CanDeactivate
- Absolute/Relative RouterLink

## How to use

Clone this github repo and run the following commands:

- npm install
- npm start
